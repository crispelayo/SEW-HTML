Elementos de encabezado h1-h6

Los elementos de encabezado h1 a h6 se utilizan para implementar seis niveles de encabezado del documento, con h1 es el más importante, y h6, el menos importante.

h1 Debería ser único en el ámbito del documento (accesibilidad) en un sitio web con varios documentos suele identificar al propio sitio web. Por ejemplo Universidad de Oviedo.

elementos de encabezado h2

El resto de los elementos deben utilizarse para marcar la estructura lógica del contenido con una secuencia lógica (si aparece h3 es porque antes hay un h2 que esta relacionado) nunca para mostrar encabezados de menor tamaño (la visualización se modifica a través de hojas de estilo)
