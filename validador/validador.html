Errores y advertencias en HTML5

Este ejemplo sirve para mostrar algunas de las advertencias que se producen en el validador de HTML del W3C. El validador se actualiza constantemente y permite además que los usuarios introduzcan comentarios para seguir mejorando su funcionamiento. Por tanto la validez de este documento se refiere a la fecha de la última edición

Ultima actualización 25/09/2015

Utilización de elementos que no se encuentran en la estándar

El elemento <hgroup> esta definido por el whatwg pero no se encuentra definido en la especificación del W3C, por lo que genera un error. Ejemplo:

Encabezado de nivel3 dentro de <hgroup>

Encabezado de nivel4 dentro de <hgroup>

Mensaje del validador: (Error) Element hgroup not allowed as child of element body in this context. (Suppressing further errors from this subtree.)

Mensaje del validador: (Error) The hgroup element is obsolete. To mark up subheadings, consider either just putting the subheading into a p element after the h1-h6 element containing the main heading, or else putting the subheading directly within the h1-h6 element containing the main heading, but separated from the main heading by punctuation and/or within, for example, a span class="subheading" element with differentiated styling. To group headings and subheadings, alternative titles, or taglines, consider using the header or div elements.

Utilización de elementos que están implementados en los navegadores

El validador avisa de que ciertas características que se están utilizando no están implementadas en todos los navegadores y eso puede provocar problemas de visualización a los usuarios.

Por ejemplo la utilización del elemento <date> en un formulario

Fecha nacimiento

Mensaje del validador: (Warning) The date input type is not supported in all browsers. Please be sure to test, and consider using a polyfill.

Utilización de elementos estructurales sin encabezados

El validador lanza una advertencia si se crean elementos estructurales por ejemplo <section>y no se incluye en ella un encabezado. Ejemplo:

Esta es una sección sin encabezado asociado

Mensaje del validador: (Warning) Section lacks heading. Consider using h2-h6 elements to add identifying headings to all sections.

Codificación incorrecta del contenido del archivo

El validador lanza una advertencia si se constata que el documento html contiene BOM al estar codificado en UTF-8. Ejemplo en el ejercicio de "textoblando.html"

Mensaje del validador: (Warning)Byte-Order Mark found in UTF-8 File. The Unicode Byte-Order Mark (BOM) in UTF-8 encoded files is known to cause problems for some text editors and older browsers. You may want to consider avoiding its use until it is better supported.
