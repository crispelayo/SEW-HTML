Ejemplos de HTML5: texto

a
Enlace a la sección de discos
em
Cuando quiero resaltar alguna parte del texto para enfatizarla de forma suave
strong
Cuando quiero resaltar alguna parte del texto para enfatizarla de forma fuerte
small
Para definir que el texto debe ser mas pequeño
s
Para tachar el texto como no relevante
cite
Para representar el titulo de una obra, trabajo, etc.
q
Para citar texto de alguna otra fuente
dfn
Para marcar la primera instancia de la defición de un término. MIW es el Máster en Ingeniería Web
abbr
Nos permite definir el significado de una abreviatura. El MIW es un Máster universitario de la Universidad de Oviedo.
data
Permite establecer un valor para representar el contenido. El MIW tiene dos cursos lectivos.
time
Permite marcar un valor de tiempo-fecha. Justo antes de medianoche 23:59 .
code
Permite marcar un fragmento como código de ordenador. El método new crea instancias.
