Caracteres especiales en HTML5

En HTML existen una serie de caracteres especiales que si es necesario utilizarlos en el contenido deben escribirse mediante entidades de caracteres XML y HTML.

  • El caracter < se establece escribiendo: &lt;
  • El caracter > se establece escribiendo: &gt;
  • El caracter " se establece escribiendo: &quot;
  • El caracter ' se establece escribiendo: &apos;
  • El caracter & se establece escribiendo: &amp;

En la Wikipedia se encuentra la lista completa de entidades de caracter XML y HTML
