Atributos en HTML5

Los atributos en HTML5 pueden ser globales que se pueden utilizar en todos los elementos y particulares de un elemento. Cada elemento HTML indica en su descripción los atributos que tiene.

Atributos Globales más habituales

accesskey
Permite a los agentes de usuario crear atajos de teclado para activar el foco del elemento. Ejemplo: <title="Actividades" accesskey="A" href= "actividades.html"> Resultado: [A]ctividades.
hidden
Permite definir contenido oculto. ATENCIÓN no debe utilizarse para no mostrar contenido del sitio. Ejemplo: <p>Este texto aparecerá visible hasta aqui, despues <span hidden>estará oculto</span></p> Resultado:

Este texto aparecerá visible hasta aqui, despues estará oculto

lang
Permite establecer el idioma que se utiliza Ejemplo: <p lang="fr"> Ce paragraphe est rédigé en français.</p>
style
Permite especificar un estilo para el elemento en formato CSS. ATENCIÓN: para garantizar la separación entre contenido y presentación debe evitarse utilizar este atributo. Ejemplo: <p>Mi chaqueta es <span style= "color: green"> verde</span> y mis ojos son <span style="color: blue"> azules</span>.</p>
Resultado: Mi chaqueta es verde y mis ojos son azules.
tabindex
Permite especificar el orden de los elementos y cuando tienen el foco de navegación.
Ejemplo: <a title="Actividades" tabindex= " 2" href= "actividades.html">Actividades</a>
<a title= "Enlaces" tabindex= " 1" href= " enlaces.html">Enlaces</a>
Resultado: Actividades Enlaces
title
Permite especificar el título de un elemento. Ejemplo: <a title="Enlace a Actividades" href= "#">Actividades</a>
Resultado: colocar el ratón sobre el enlace Actividades
dir
Permite indicar la direccionalidad del texto.
Ejemplo: <p dir="rtl">حولا</p> Resultado:

حولا

Ejemplo: <p dir="rtl">hola</p> Resultado:

hola

translate
Permite indicar si el elemento será traducido cuando el documento sea internacionalizado o no.

Otros atributos

id
permite asignar un identificador único a un elemento Ejemplo: <section id="principal">
class
Permite clasificar a los elementos de HTML en clases que comparten características. Ejemplo <p class="destacado">
Atributo href
permite indicar la ruta donde se encuentra un documento o una ubicación de internet <a href="estructura/address.html"> ó <a href="http://www.google.com">
Atributo src
permite indicar la ruta donde se encuentra un archivo para acceder a él <img src="multimedia/trailer.mp4">
Atributo download
obliga a realizar la descarga de un recurso enlazado <a href="plantilla.html" download>
Atributo reversed
en una lista numerada la ordena en sentido inverso <ol reversed>

Activación de los atajos de Teclado:

  • Safari: ALT+atajo
  • Microsoft Edge: ALT+atajo+ENTER
  • Firefox: MAY+ALT+atajo
  • Opera: MAY+ESC+atajo
  • Chrome: ALT+atajo
